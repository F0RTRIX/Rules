• 1. Общее положение •
1.1 - Устав является общепринятым и обязателен для исполнения всеми сотрудниками МВД.
1.2 - Несоблюдение устава карается - эмбарго / увольнением / понижением / выговором / беседой.
1.3 - Незнание устава карается увольнением.
1.4 - Набор в полицию осуществлять только при наличии 3 лет проживания в штате!


• 2. Основные обязанности •
2.1 - Знать УК, АК , Устав МВД в обязательном порядке.
2.2 - Поимка преступников на территории города и прилегающих к городу территорий.
2.3 - Исполнять приказы старших по званию.
2.4 - Обеспечение безопасности жизнедеятельности граждан.
2.5 - Уважительное отношение к гражданам и к своим сотрудникам
2.6 - Борьба с нарушениями правил дорожного движения .
2.7 - При необходимости,показать документы утверждающие личность и принадлежность к Полиции/ФБР (Значок).
2.8 - Изъятие запрещенных предметов и предотвращение их перевозки.

• 3. Сотрудникам МВД запрещено •
3.1 - Употребление алкоголя и наркотических средств.
3.2 - Принимать взятки.
3.3 - Открывать огонь по гражданским. Исключение: угроза жизни офицера/агента.
3.4 - Пользоваться ограждениями, знаками и т.д. без причины (( /break )).
3.5 - Нарушать правила штата.
3.6 - Посещать развлекательные мероприятия в рабочее время.
3.7 - Брать транспорт не по званию/должности.
3.8 - Материться, флудить, оскорблять кого-либо в рацию.
3.9 - Разговаривать на построении.
3.10 - Доставать оружие на построении.
3.11 - Доставать мобильный телефон во время построения.
3.12 - Спать на построении.
3.13 - Стрелять на построении.
3.14 - Выходить из строя без разрешения.
3.15 - Разговаривать в рацию на любые темы не касающиеся работы.
3.16 - Покидать город в рабочее время. Исключение: погоня.
3.17 - Покидать пост/патруль/заканчивать работу без доклада в рацию.
3.18 - Спать на посту.
3.19 - Стоять в гараже без дела.
3.20 - Работать в рабочее время на шахте и на заводе.
3.21 - Находится на территории казино, а тем более играть там в рабочее время.
3.22 - Показывать гражданским свой паспорт. Разрешено жетон.
3.23 - Заезжать в опасный район.
3.24 - Нарушать субординацию.

• 4. Сотрудникам МВД разрешено •
4.1 - Требовать от граждан и должностных лиц прекращения административного или уголовного правонарушения.
4.2 - Доставлять правонарушителей в отделение полиции ( Гараж / КПЗ ).
4.3 - Проверять у граждан и должностных лиц документы,удостоверяющие личность.
4.4 - Доставлять в отделение полиции лица,подозреваемые в совершенном преступлении.
4.5 - Лишать вод.прав за грубое нарушение Правил Дорожного Движения.
4.6 - Осуществлять обыск граждан в гараже.
4.7 - Останавливать Т.С. во время движения для проверки документов , а так же для проверки документов на автомобиль.
4.8 - Оцеплять местность для проведения операций по устранению массовых беспорядков.
4.9 - Подрабатывать в не рабочее время на предприятиях штата.
4.10 - Сотрудник МВД имеет право выгнать с территории МВД сотрудника СМИ, МЗО, МО, АП, Мэрии если отсутсвует разрешение лидера на его присутствие.
4.11 - Сотрудник МВД в праве нарушить правила дорожного движения если того требует ситуация для выполнения его служебный обязанностей
4.12 - Высший состав ФБР, включая Старшего Агента, вправе отдать приказ сотруднику Полицейского Департамента.

• 5. Сотрудник МВД обязан •
5.1 - Пресекать уголовные правонарушения.
5.2 - Оказывать помощь гражданам,пострадавшим от преступной деятельности.
5.3 - При обнаружении запрещенных предметов у подозреваемого, забрать предметы.
5.4 - Обеспечивать правопорядок на улицах города,безопасность граждан и/или должностных лиц.
5.5 - Действовать в рамках УК и АК штата.
5.6 - Докладывать о происшествиях в рацию.В некоторых случаях вызывать помощь.
5.7 - Объективно оценивать обстановку вокруг себя/на посту.Быть всегда готовым к любому происшествию.
5.8 - При получении информации о предстоящем преступлении немедля сообщать в рацию.
5.9 - Выявлять и пресекать наркотический оборот/продажу оружия в штате.
5.10 - Знать основные положения Правил Дорожного Движения.
5.11 - Предупреждать и пресекать нарушения Правил Дорожного Движения.
5.12 - Быть вежливым и тактичным в обращении с гражданами и/или должностными лица.
5.13 - Покидая пост доложить в рацию.

• 6. Коды • 
Ситуационные коды?
CODE 0 - Необходима немедленная поддержка. Подается, когда офицер ранен/Чрезвычайное положение в штате, теракт.
Сотрудники не имеют права не реагировать на данный сигнал.
CODE 1 – Офицер полиции в бедственном положении и ему требуется немедленная поддержка. Например обстреливают его.
CODE 2 - Вызов (без сирен/стробоскопов)
CODE 3 - Срочный вызов (с сиренами/стробоскопами, приоритетней C'2)
CODE 4 - Помощь не требуется.
CODE 4 ADAM [TIME] - Объявляется в случае утери визуального контакта с подозреваемым, указывается время действия данного кода и приоритетные районы поиска.
CODE 5 - Требуется помощь офицеров.
CODE 6 - Задерживаюсь на (Локация).
CODE 7 - Перерыв на обед. (АФК)

Десятичные коды?
CODE 10-4. Принято.
CODE 10-6. Не принято.
CODE 10-15. Подозреваемые арестованы (включая кол-во подозреваемых, локацию. При оформлении преступника в участок).
CODE 10-20. Локация. (Район и улица)
CODE 10-21. Запрос ситуации и местонахождения. (Для кураторов патрульного дивизиона)
CODE 10-40. Большое скопление людей. (В 2 раза больше, чем МВД на ситуации. Реагирование по 3 коду)
CODE 10-55. Traffic-Stop
CODE 10-57 VICTOR. - Погоня за автомобилем (включая модель авто, цвет, местонахождение, направление движения и сообщение о переходе в Discord).
CODE 10-57 FOXTROT. - Пешая погоня (включая внешность подозреваемого, оружие (если имеется), местонахождение, направление движения, Discord)
CODE 10-60. Информация об автомобиле. (Включая цвет, марку, количество людей внутри(расу))
CODE 10-61. Информация о пешем подозреваемом. (Включая расу, одежду.)
CODE 10-66. Остановка повышенного риска.
CODE 10-99. Ситуация урегулирована.
